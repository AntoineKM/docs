# Ignite

Ignite is the pragmatic app deployment platform: deploy and scale any application worldwide, with just a Dockerfile.

Ignite provides you with a gradual approach to deploying your apps: it's easy to deploy a simple app, but there are powerful features which you can easily add on when you need them.

When designing Ignite, we decided to prioritize 3 things:
- Reliability,
- Performance, and
- Developer experience
